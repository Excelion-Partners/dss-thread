## Install Issues?

If you see the following error:

```bash
npm ERR! code ERESOLVE
npm ERR! ERESOLVE could not resolve
```

Try running:

```bash
npm install --legacy-peer-deps
```

## To build React application

```bash
cd ./resource/thread/
git pull & npm run-script build
```
